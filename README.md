# Password Generator

## Project Description

This is a simple password generator built using HTML, CSS, and JavaScript. It lets users generate random passwords by choosing the password length and the types of characters they want to include, such as uppercase letters, lowercase letters, numbers, and special characters. It also shows the strength of the generated password and allows it to be copied with a single click.

## Technologies Used

* HTML
* CSS
* JavaScript

## Steps to Run the Project

1. Clone or download this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
4. Select the password options and click **Generate Password**.

No installation or additional setup is required.
