# Password Generator

A basic password generator made using HTML, CSS, and JavaScript. It generates secure random passwords based on the options selected by the user.

## Features

* Generate random passwords
* Select password length
* Include uppercase letters
* Include lowercase letters
* Include numbers
* Include special characters
* Copy password to clipboard
* Password strength indicator

## Technologies Used

* HTML
* CSS
* JavaScript
